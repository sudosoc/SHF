# SHF - sudosoc Hybrid Framework

SHF (sudosoc Hybrid Framework) is a modular cyber security framework that combines:

- Offensive modules (Red Team)
- Defensive modules (Blue Team)
- DFIR / Forensics
- Threat Intelligence

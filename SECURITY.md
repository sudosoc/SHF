# Security Policy - SHF (SudoSoc Hybrid Framework)

SHF is a cybersecurity framework intended **ONLY** for:

- Educational use
- Authorized penetration testing
- Research on systems and networks you own or have explicit written permission to test

Any use of SHF for:
- Unauthorized access
- Illegal hacking
- Attacks against third-party systems

is strictly prohibited and is the sole responsibility of the user.

sudosoc does **NOT** accept any liability for misuse of this framework.
